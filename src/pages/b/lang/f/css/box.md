### 2. Padding
パディングの大きさの設定の仕方は2通り。
1. 四辺ごとに設定する
```css:style.css
* {
    padding-top: ;
    padding-right: ;
    padding-left: ;
    padding-bottom: ;
    }
```

2. 四辺纏めて設定する
```css:style.css
* {
    margin: TOP RIGHT BOTTOM LEFT;
    }
```
